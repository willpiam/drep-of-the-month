# dRep Of The Month

Highlighting one dRep and one SPO every month.

## Disclaimer

I do not claim to be unbiased. In fact, take it for granted that this list is biased, I still think these are good options, but what do I know? I'm biased. 

These entries are subject to change. If you have a problem with how I run this I suggest you fork it and do things how you like. If you look into `schedule.json` you can see who is coming up. This is subject to change. 

## Generate Schedule

    python3 generate_schedule.py

## Extend Schedule

Extend the existing schedule by N months

    python3 generate_schedule.py --continue 10
